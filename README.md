
## 📖 简介 (Introduction) [English](README_EN.md) | [中文](README.md)

**UcardPay** 是加密货币发卡与支付管理系统。它旨在帮助快速搭建自己的U卡分发平台，支持Visa/Mastercard虚拟卡/实体卡批量开户、USDT 自动充值、法币实时结算以及完整的代理商分销体系。
类似于THPAY / WildCard 等U卡虚拟卡项目

## ✨ 核心特性 (Features)

- 💳 **虚拟卡管理**：支持 Visa/Mastercard 虚拟卡的实时申请、激活、冻结与注销。
- 💰 **多币种钱包**：原生支持多币种USDT/ETH/BTC等50多种加密货币 (TRC20/ERC20等多区块链) 充值，自动汇率换算为 USD/EUR等20种法币。
- 🔄 **自动结算**：对接交易所流动性接口，实现 USDT -> 法币的毫秒级兑换。
- 📊 **代理商系统**：支持无限级代理模式，自定义开卡费、充值手续费返佣比例。
- 🛡️ **风控系统**：内置 KYC (Know Your Customer) 验证流程与高风险交易拦截（可取消KYC）。
- 🔌 **开发者 API**：提供完善的 API 文档，方便第三方商户接入发卡能力。
- 充值方式：支付宝、微信、USDT，无需KYC、无年费，无最低充值金额

## 🛠️ 技术栈 (Tech Stack)

- **后端框架**: ThinkPHP 6.0
- **数据库**: MySQL 5.7 / 8.0
- **缓存**: Redis (用于队列与高频配置)
- **运行环境**: PHP 8.0+ (推荐 8.1), Nginx
- **加密库**: OpenSSL (RSA/AES)
